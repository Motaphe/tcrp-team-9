# Postman Test URLs

## Events

- GET all events: `GET http://localhost:3000/api/events`
- GET specific event: `GET http://localhost:3000/api/events/:id`
- CREATE new event: `POST http://localhost:3000/api/events`
- UPDATE event: `PATCH http://localhost:3000/api/events/:id`
- DELETE event: `DELETE http://localhost:3000/api/events/:id`
